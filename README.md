# Dota 2 Timings HUD

## 1) Что это за проект
Это небольшой HUD (окно поверх игры), который показывает тайминги и напоминания по макро‑событиям Dota 2: руны, лотос, терзатель, окна риска и т.д. HUD нужен, чтобы вовремя принимать решения и не пропускать важные моменты матча.

**Важно:**
- HUD использует **официальный Game State Integration (GSI) от Valve**.
- HUD **не вмешивается в игровой процесс** и не изменяет файлы игры.
- Это **не чит** и не дает скрытую информацию: он отображает только заранее заданные тайминги и общую информацию.

## 2) Системные требования
- **Windows 10/11**
- Установленная **Dota 2**
- **Python 3.10–3.12**
- Права администратора **не требуются**

## 3) Установка
### Шаг 1. Установить Python (если не установлен)
1. Перейдите на сайт https://www.python.org/downloads/
2. Скачайте Python 3.10 или новее.
3. При установке **обязательно отметьте галочку** “Add Python to PATH”.

Проверка установки:
```bash
python --version
```

### Шаг 2. Скачать проект
Скачайте архив проекта с GitHub и распакуйте его в удобную папку, например:
```
C:\Games\dota-hud
```

### Шаг 3. Установить зависимости
Откройте **Командную строку** в папке проекта и выполните:
```bash
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

Если у вас нет `requirements.txt`, установите зависимости вручную:
```bash
python -m pip install PyYAML keyboard PySide6
```

## 4) Настройка Dota 2 (GSI)
**Обязательно** создать папку:
```
<папка_steam>\steamapps\common\dota 2 beta\game\dota\cfg\gamestate_integration\
```

Пример полного пути:
```
C:\Program Files (x86)\Steam\steamapps\common\dota 2 beta\game\dota\cfg\gamestate_integration\
```

Создайте файл конфигурации, например:
```
dota_hud.cfg
```

Пример содержимого файла:
```
"Dota HUD"
{
  "uri" "http://127.0.0.1:4000"
  "timeout" "5.0"
  "buffer" "0.1"
  "throttle" "0.1"
  "heartbeat" "30.0"
  "data"
  {
    "map" "1"
  }
}
```

После этого **перезапустите Dota 2**.

## 5) Запуск HUD
В командной строке в папке проекта:
```bash
python -m dota_hud
```

Что должно появиться:
- Полупрозрачное окно HUD поверх всех окон.
- В строке NOW появится сообщение **“SYNCED WITH DOTA”**, когда данные из игры поступают.

Как проверить, что GSI работает:
- Запустите матч (обычный или тренировочный).
- В HUD должна обновляться строка таймера.
- В консоли (окне командной строки) будут появляться сообщения `[GSI] ...`.

## 6) Управление HUD
Горячие клавиши (по умолчанию):
- **F8** — старт таймера
- **F9** — стоп таймера
- **F10** — сброс таймера
- **F7** — lock/unlock окна (клики сквозь HUD)

Поведение lock/unlock:
- **Lock**: окно закреплено, клики проходят **сквозь HUD** (можно кликать в игре).
- **Unlock**: окно можно **перемещать мышью**.

Закрыть HUD можно, закрыв окно командной строки или нажав **Ctrl+C** в консоли.

## 7) Что отображает HUD
- **Таймер матча**
- **Предупреждения и окна риска**
- **Макро‑информация**: руны, лотос, терзатель, важные временные окна

HUD **не “знает” действия врагов** и не показывает скрытую информацию — только тайминги и напоминания.

## 8) Безопасность и легальность
- HUD использует **официальный GSI** от Valve.
- HUD **не вмешивается в клиент Dota 2** и не изменяет игровые файлы.
- HUD **не дает скрытых данных** и **не автоматизирует действия**.

Поэтому HUD не является читом и не дает unfair advantage.

## 9) Частые проблемы
### HUD не появляется
- Проверьте, что вы запускаете команду из папки проекта.
- Убедитесь, что Python установлен и доступен в PATH.

### Dota не шлёт данные
- Проверьте путь к папке `gamestate_integration`.
- Убедитесь, что файл `.cfg` создан и имеет правильное содержимое.
- Перезапустите Dota 2.

### Нет синхронизации
- Убедитесь, что матч действительно запущен.
- Проверьте, что в консоли HUD появляются строки `[GSI]`.

## 10) Ограничения проекта
- HUD **не показывает действия врагов**.
- HUD **не автоматизирует действия**.
- HUD работает **только во время матча** и отображает лишь тайминги.

## Выбор интерфейса (Tkinter / PySide6)
Проект поддерживает два варианта UI:
- **Tkinter (legacy)** — работает везде, fallback по умолчанию.
- **PySide6 (новый HUD)** — полупрозрачный градиентный фон, стиль ближе к Dota+.

Чтобы выбрать UI, откройте файл конфигурации (например `configs/timings.yaml`) и задайте:
```yaml
ui:
  backend: "qt"   # для PySide6
  # backend: "tk" # для Tkinter (legacy)
```

Если PySide6 не установлен, используйте `backend: "tk"` (или удалите блок `ui`, тогда будет Tkinter).
